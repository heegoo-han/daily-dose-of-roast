## Component Inventory

| # | Component | Type | Content/State |
|---|-----------|------|---------------|
| ① | 사용자 아바타 + 다크모드 토글 | Avatar, Button (ghost) | `(JD)` 이니셜 아바타, `☀` 아이콘 버튼 |
| ② | 검색 입력 | Input | placeholder: "검색...", 실시간 필터 |
| ③ | 우선순위 필터 | Select | options: 전체, High, Medium, Low |
| ④ | 태그 필터 | Select | options: 전체, 디자인, 백엔드, 프론트엔드, 문서 |
| ⑤ | 칸반 카드 | Card | 제목 + 우선순위 배지 + 날짜 + 태그 칩 |
| ⑥ | 우선순위 배지 | Badge (destructive) | `(● High)` 빨강, `(Medium)` 노랑, `(Low)` 초록 |
| ⑦ | 새 카드 제목 입력 | Input | placeholder: "새 작업..." |
| ⑧ | 카드 추가 버튼 | Button (default) | label: "+ 추가" |

## Interactions

- [②] 검색어 입력 → 제목에 검색어 포함된 카드만 실시간 표시
- [③] 우선순위 선택 → 해당 우선순위 카드만 필터링
- [④] 태그 선택 → 해당 태그를 가진 카드만 필터링
- [⑤] 카드 드래그 → 다른 칼럼으로 이동, 같은 칼럼 내 순서 변경 가능
- [⑦⑧] 제목 입력 + 추가 클릭 → Todo 칼럼에 새 카드 생성 (기본 Medium)
- [⑧] 빈 제목으로 추가 시도 → 카드 생성되지 않음, 에러 피드백
- [①] ☀ 클릭 → 다크모드 전환, localStorage 저장

## Notes

- 칼럼은 Todo / In Progress / Done 3개 고정
- 카드 순서: 최신 추가 카드가 하단에 위치
- 드래그 앤 드롭은 ASCII로 표현 불가 — Interactions에서 기술
- 다크모드는 레이아웃 동일, 다크 테마 적용 (별도 와이어프레임 불필요)
