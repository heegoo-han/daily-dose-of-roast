## Component Inventory

| # | Component | Type | Content/State |
|---|-----------|------|---------------|
| ① | 검색 입력 | Input | value: "버그", 포커스 상태 (이중선 테두리) |
| ② | 우선순위 필터 | Select | value: "High", 활성 상태 (이중선 테두리) |
| ③ | 활성 필터 칩 | Badge (secondary) | `(버그) ✕`, `(● High) ✕` — 각각 ✕로 개별 해제 가능 |

## Interactions

- [①] 검색어 수정/삭제 → 실시간으로 필터 결과 갱신, 별도 버튼 없음
- [②] 우선순위 변경 → 필터 결과 즉시 갱신
- [③] 필터 칩 ✕ 클릭 → 해당 필터 해제, 결과 갱신
- [③] 모든 필터 해제 → 전체 카드 다시 표시

## Notes

- 검색 "버그" + 필터 "High" = AND 조건: "버그" 포함 + High 우선순위인 카드만 표시
- 조건에 맞는 카드가 없는 칼럼은 점선 테두리 빈 상태 메시지 표시
- 활성 필터가 있는 Input/Select는 이중선 테두리(`══`)로 강조
- 필터링은 실시간 — 타이핑 즉시 반영
- Todo (1): "버그 수정 #42"만 필터 통과 (제목에 "버그" 포함 + High 우선순위)
