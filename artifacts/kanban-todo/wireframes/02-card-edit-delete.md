## Component Inventory

| # | Component | Type | Content/State |
|---|-----------|------|---------------|
| ① | 카드 제목 편집 | Input | value: "수정된 작업", 포커스 상태 |
| ② | 우선순위 셀렉트 | Select | value: "High", options: High/Medium/Low |
| ③ | 태그 칩 | Badge (secondary) | 기존 태그 표시 + `✕` 제거 버튼 |
| ④ | 태그 추가 입력 | Input | placeholder: "태그 추가..." |
| ⑤ | 저장 버튼 | Button (default) | label: "저장" |
| ⑥ | 삭제 버튼 | Button (ghost) | 🗑 아이콘, 클릭 시 ⑦ 다이얼로그 표시 |
| ⑦ | 삭제 확인 다이얼로그 | AlertDialog | 카드명 포함 확인 메시지 |
| ⑧ | 취소 버튼 | Button (outline) | label: "취소", 다이얼로그 닫기 |
| ⑨ | 삭제 확인 버튼 | Button (destructive) | label: "Delete x", 카드 영구 삭제 |

## Interactions

- [①] 제목 수정 → 즉시 반영, 빈 문자열 입력 시 변경 거부 + 이전 제목 유지
- [②] 우선순위 변경 → 카드 색상 즉시 변경 (High=빨강, Medium=노랑, Low=초록)
- [③] ✕ 클릭 → 해당 태그 제거
- [④] 태그 텍스트 입력 + Enter → 새 태그 칩 추가
- [⑤] 저장 클릭 → 편집 모드 종료, localStorage 반영
- [⑥] 🗑 클릭 → ⑦ 삭제 확인 다이얼로그 표시
- [⑧] 취소 클릭 → 다이얼로그 닫기, 카드 유지
- [⑨] Delete 클릭 → 카드 삭제, localStorage 반영

## Notes

- 편집 모드 카드는 이중선 테두리(`══`)로 활성 상태 강조
- 삭제 다이얼로그는 `░░` 오버레이 위에 중앙 배치
- 편집과 삭제가 동시에 표시되지만, 실제 사용 시 두 상태는 독립적
